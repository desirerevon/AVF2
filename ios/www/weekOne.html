Visit Programmableweb.com and search for JSON-P data sources. Why would we need JSON-P, specifically?

Well first let’s get the difference out of the way for less confusion. JSON is the data format. JSON-P is the method wrapped around it that is used during parsing. When you are trying to pull data from another site such as Twitter and you are trying to pull specific data in order to cross browse and retrieve specific data from that page you need to use a callback function or method. If you call a JSON object without a function you will get plain data back. If you retrieve data from the same site you can use regular JSON. So if you want certain data you need to use the callback function to specify what you want returned.

Briefly describe some past 'lessons learned' regarding workflow as it relates to research, design, development, etc.

The way I began my applications is brainstorming. It’s hard sometimes because I have a lot of ideas, but the design aspect would take me longer than the assignments.
